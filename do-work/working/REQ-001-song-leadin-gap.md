# Add Lead-in Gap Before First Song Note

## What
Add a gap/rest before the first note of every song so the player has time to prepare after the 3-2-1 countdown ends. The bars should not start immediately at the play zone — there should be visible empty space scrolling first.

## Context
Currently the first target bar appears immediately when the song engine starts after the countdown. The player needs a moment to see the first note approaching before they need to hit it.
